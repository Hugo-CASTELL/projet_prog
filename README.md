## Lancement du menu de commandes

Une procédure automatique de lancement du menu de commmandes a été développé.

Se placer dans le dossier de tests :
```
cd ./src/menu/
```

Lancer le script de tests :
```
chmod +x ./run_menu.sh && sh ./run_menu.sh
```
