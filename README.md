## Lancement des tests

Une procédure automatique de lancement des tests a été développé.

Se placer dans le dossier de tests :
```
cd ./src/tests/
```

Lancer le script de tests :
```
chmod 777 ./run_tests.sh && ./run_tests.sh
```
